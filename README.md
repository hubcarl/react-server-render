#react-server-render
CNode社区React服务器端渲染

#Run#
 1. clone代码到本地
 2. cd react-server-render
 3. npm install
 4. grunt 
 5. node bin/www 或者 npm start
 6. http://localhost:4000
 

#重要的框架或类库#
 - 后端渲染引擎 [react-engine](https://github.com/paypal/react-engine)
 - 路由管理 [react-router](https://github.com/rackt/react-router)
